# Security Policy

## Supported Versions

The project is pre-1.0.0. The `main` branch is the only supported version.

## Reporting a Vulnerability

Do not open public GitHub issues for security vulnerabilities. Report security concerns via the contacts listed below.

- Email: security@pixellabsai.app
- Private advisory: use GitHub security advisories

## What to Include

- Description of the issue
- Steps to reproduce
- Impact assessment
- Suggested remediation (if available)

## Response Targets

- Initial response: within 48 hours
- Status update: within 7 days
- Resolution target: based on severity

## Security Best Practices

- Never commit secrets
- Use environment variables for API keys
- Rotate keys on leakage
- Keep dependencies up to date

## Contact

For security concerns: security@pixellabsai.app
